## Key Differences from Base Code
### 1. N_VA_StaticAbsolute_color.py
This python script is adapted form the script "N_VA_StaticAbsolute.py" that visualizs the moving object in a Static Absolute view. The new version, "N_VA_StaticAbsolute_color.py", enhances the visualization by:  
- using color gradients to represent temporal order,  
- using arrow sizes to indicate movement amplitude (or speed), and  
- marking stationary points with red dots.
